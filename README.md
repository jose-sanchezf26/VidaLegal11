# Asesoría VidaLegal11 - Web App (FastAPI + HTML/CSS + PostgreSQL)

Proyecto real de una web corporativa para la asesoría VidaLegal11, desarrollada con FastAPI, Jinja2, HTML/CSS, PostgreSQL y una arquitectura limpia y escalable.

El objetivo es ofrecer:
- Página informativa de servicios y contacto.
- Formulario de contacto que almacena mensajes en PostgreSQL.
- Base para añadir autenticación, dashboard interno y administración en el futuro.

## Tecnologías utilizadas
- **Backend**: FastAPI
- **Frontend**: Jinja2 + HTML + CSS
- **Bases de datos**: PostgreSQL (con SQLAlchemy o psycopg)
- **Servidor**: Uvicorn
- **Estructura modular**: routers, models, services, utils
